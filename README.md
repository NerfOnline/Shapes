### Commands
The following commands may be used.

    /shapes                      -- brings up the main window
    /shapes clear                -- this will clear all markers currently placed
    /shapes <marker_name>        -- this will toggle specified marker between on or off
    /shapes <marker_name> on|off -- this will turn specified marker on or off
